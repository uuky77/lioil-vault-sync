# Lioil Vault Sync

Obsidian 多云端仓库同步插件。支持将你的 vault 同步到多种云存储服务。

## 支持的云服务

- **百度网盘**
- **阿里云盘**
- **夸克网盘**
- **WebDAV**（Nextcloud、ownCloud 等）
- **Amazon S3** 兼容（AWS S3、MinIO、Cloudflare R2 等）
- **Google Drive**
- **OneDrive**

## 功能特性

### 同步引擎

- **双向同步** — 本地与云端互相同步，而非单向覆盖
- **增量比对** — 基于 mtime + md5 的差异检测，只传输有变化的文件
- **冲突检测** — 本地和云端同时修改时自动识别冲突
- **冲突策略** — 支持四种处理方式：询问用户 / 保留本地 / 保留云端 / 保留两者
- **删除同步** — 本地删除同步云端、云端删除同步本地
- **移动检测** — 基于文件 md5 追踪重命名和移动操作
- **文件夹同步** — 自动创建父目录，完整支持目录结构

### 安全与隐私

- **加密同步** — 文件上传前可选 AES 加密
- **OAuth 2.0** — 百度、阿里、Google 等支持标准 OAuth 登录
- **本地存储** — Token 仅存储在 Obsidian 本地，不经过第三方服务器
- **零追踪** — 无分析、无遥测、无数据收集

### 自动化

- **定时自动同步** — 可配置同步间隔（最短 1 分钟）
- **命令面板** — `Ctrl+P` → "立即同步" 一键触发
- **文件过滤** — 支持 glob 规则排除文件、文件大小限制
- **选择性同步** — 可勾选特定文件/文件夹进行部分同步

### 其他

- **MCP 服务器** — 内置 MCP 协议支持，可被 AI 代理调用
- **中英文界面** — 完整 i18n 支持
- **存储信息** — 显示云盘用量

## 安装

### 从 Obsidian 社区插件市场

1. 打开 设置 → 第三方插件
2. 搜索 "Lioil Vault Sync"
3. 安装并启用

### 手动安装

1. 从 [Releases](https://github.com/uuky77/lioil-vault-sync/releases) 下载 `main.js`、`manifest.json`、`styles.css`
2. 放入 vault 的 `.obsidian/plugins/lioil-vault-sync/` 目录
3. 重启 Obsidian 并在设置中启用

## 使用指南

1. 在 设置 → 第三方插件 中启用插件
2. 点击左侧功能区的云朵图标，或运行 `Ctrl+P` → "Open Cloud Vault Sync"
3. 选择云服务商，通过 OAuth 授权登录
4. 在插件设置中配置同步目录和其他选项
5. 开启自动同步或手动点击"全部同步"

## 同步场景对照

| 场景 | 本地 | 云端 | 结果 |
|------|------|------|------|
| 新建文件 | 有 | 无 | 上传至云端 |
| 云端新增 | 无 | 有 | 下载至本地 |
| 本地修改 | 新于云端 | 旧于本地 | 上传覆盖云端 |
| 云端修改 | 旧于云端 | 新于本地 | 下载覆盖本地 |
| 同时修改 | 已改动 | 已改动 | 按策略处理冲突 |
| 本地删除 | 已删除 | 存在 | 同步删除云端 |
| 云端删除 | 存在 | 已删除 | 同步删除本地 |
| 文件移动 | 新路径 | 旧路径 | md5 检测后重命名 |

## 第三方服务

本插件连接到以下第三方 API：

| 服务商 | 域名 | 用途 |
|--------|------|------|
| 百度网盘 | `pan.baidu.com`、`d.pcs.baidu.com` | 文件上传下载、OAuth |
| 阿里云盘 | `auth.aliyundrive.com`、`api.aliyundrive.com` | 文件同步、OAuth |
| Google Drive | `www.googleapis.com`、`oauth2.googleapis.com` | 文件同步、OAuth |

所有数据直接在您的设备和云服务商之间传输，不经过任何中间服务器。
