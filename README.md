# Lioil Vault Sync

Multi-cloud vault sync plugin for Obsidian. Supports Baidu Netdisk, Aliyun Drive, WebDAV, S3 and more.

## Installation

Copy main.js, manifest.json, and styles.css into your vault's .obsidian/plugins/lioil-vault-sync/ directory, then enable the plugin in Obsidian's Community Plugins settings.

## Supported Cloud Providers

- Baidu Netdisk
- Aliyun Drive
- Quark Drive  
- WebDAV
- Amazon S3

## Features

- Cloud file browsing & management
- MCP server for AI agent integration
- OAuth authentication
- File encryption
